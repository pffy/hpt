## INSTALL

> **NOTE:** Each method only affects one spreadsheet. 

  + Method 1:
    + [Copy this spreadsheet][copy]. (recommended)
      - You can preview the source code.
      - You get your own personal copy of HanyuPinyinTools.
  + Method 2: 
    + Visit [latest release][latest]. 
    + Download zip file. 
    + Copy and paste contents of both `addon.gs` and `data.gs` into your Apps Script project in Google Sheets.
